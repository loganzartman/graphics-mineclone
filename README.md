# graphics-mineclone
a clone of everyone's favorite voxel-based game
- procedural, "infinite" world
- entirely procedural textures
- water with screen-space refraction
- walk around

![demo](./mineclone_demo.gif)

# controls
* mouse - look
* WASD - move
* space - jump
* ctrl+f - toggle gravity
* shift - super jump

# dependencies
* OpenGL 3.3 support
* CMake 3.10
* PkgConfig
* glfw3
* glm

# building
* `make` to build
* `make run` to run
